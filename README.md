##2025 Samar's TwoFCS

This repository hold the templates used for Samar's Two FCS 2025 Conference Abstract.


# Instructions
1. Update your work.
2. Building
- To build on a Linux machine run ``make``.
- To build on a Windows machine run
  - ``pdflatex main.tex``
  - ``bibtex main.tex``
  - ``pdflatex main.tex``
  - ``pdflatex main.tex``
3. To get rid of built files, ``make clean.``
